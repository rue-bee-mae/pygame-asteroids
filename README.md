# pyame-asteroids

An asteroids clone made using Pygame as a boot.dev project.
WASD to rotate/move, space to shoot.

## Currently Implemented

* Player movement
* Shooting
* Asteroids (wow)
* Collision
  * Game over on touch asteroid
  * Split/destroy asteroid when shot
* Scoring
